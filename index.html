var HEADERS = ['商品名','品番','購入先','サイズ','点数', '','仕入日','発送日','取引完了日', '販売価格','仕入額']; function doPost(e) { try { var payload = JSON.parse(e.postData.contents); var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (payload.action === 'append') { ensureHeader(sheet); (payload.items || []).forEach(function(item) { sheet.appendRow(toRow(item)); }); return res({ok:true, inserted:(payload.items||[]).length}); } return res({ok:false, error:'unknown action'}); } catch(err) { return res({ok:false, error:err.toString()}); } } function ensureHeader(sheet) { if (sheet.getLastRow() === 0) { sheet.appendRow(HEADERS); sheet.getRange(1,1,1,HEADERS.length) .setFontWeight('bold') .setBackground('#007AFF') .setFontColor('#ffffff'); } } function toRow(item) { var q = Number(item.quantity) || 1; var c = Number(item.costPrice) || 0; var s = Number(item.salePrice) || 0; return [ item.name, item.code, item.source, item.size, q, '', item.purchaseDate||'', item.shippingDate||'', item.completionDate||'', s, c ]; } function res(obj) { return ContentService .createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }
